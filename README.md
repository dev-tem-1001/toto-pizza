ToTo-Pizza

Entity:
* Pizza (html)
* IngredientRef
* Ingredient (html)
* PizzaOrder (html)

Controller:

* Встречаем гостя и показываем пиццы (HomeController)
  * Лицевая обложка
  * Готовые варианты пиццы
  * Добавление их в заказ
  * Либо сделать свою пиццу

* Готовим пиццы (PizzaController)
  * показываем все ингредиенты
  * добавление одного или нескольких однотипных ингредиентов
  * проверка совместимости либо тонкого либо традиционного теста
  * добавление в заказ

* Создаем заказы (OrderController)
  * показ всех пицц которые в заказ
  * возможность удаление пиццы из заказа
  * возможность добавление в заказ пиццы
